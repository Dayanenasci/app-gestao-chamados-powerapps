# App de Gestão de Chamados com Power Apps

Projeto conceitual de aplicativo corporativo desenvolvido na Power Platform para gestão de chamados internos.

## Objetivo

Facilitar a abertura e o acompanhamento de solicitações técnicas por meio de uma aplicação low-code integrada com automação e IA.

## Componentes da Solução

- Power Apps (interface do usuário)
- SharePoint (base de dados)
- Power Automate (fluxo de automação)
- IA estilo Copilot (apoio na descrição técnica)# app-gestao-chamados-powerapps
